# Exchange Online Request Triage

## Common Requests
- Shared mailbox access
- Distribution list membership
- Calendar permissions

---

## Analyst Responsibilities
- Confirm request legitimacy
- Validate approvals
- Follow documented procedures
- Route elevated changes appropriately

---

## Security Considerations
- Access aligns with job function
- Least privilege enforced
- Changes documented in ticketing system

---

## Audit & Compliance
Exchange access changes often appear in:
- access reviews
- internal audits
- SOX/HIPAA evidence requests

Accurate documentation matters.
